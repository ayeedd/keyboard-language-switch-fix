\# Keyboard Language Switch Fix



Fix for keyboard language switching issues (Right Alt + Shift)

across Windows, Linux, and macOS.



---



\## العربية

\*\*المشكلة:\*\*  

اختصار تغيير لغة لوحة المفاتيح (Right Alt + Shift) لا يعمل أو يعمل بشكل غير متوقع.



\*\*السبب الشائع:\*\*  

تعارض في تعيين مفاتيح لوحة المفاتيح على مستوى النظام.



\*\*الأنظمة المدعومة:\*\*  

\- Windows 10 / 11  

\- Linux  

\- macOS  



---



\## English

\*\*Problem:\*\*  

Keyboard language switch shortcut (Right Alt + Shift) not working.



\*\*Common Cause:\*\*  

System-level keyboard key mapping conflicts.



\*\*Supported OS:\*\*  

\- Windows 10 / 11  

\- Linux  

\- macOS
